# SciPlot - 极简科研绘图工具

SciPlot 是一款专为科研人员打造的轻量级、响应式在线绘图工具。它结合了极简的设计、智能的图表推荐以及符合国际期刊标准的绘图能力。

🔗 **在线体验**: [https://sciplot.vercel.app/](https://sciplot.vercel.app/)

---

## ✨ 核心特性

- 🧠 **AI 快捷指令**: 面板顶部内置 AI 指令框。输入 `标题: 我的图表`、`science配色`、`切换趋势线` 或 `x轴: 时间(h)` 即可自动完成配置。
- 🏛️ **期刊级样式预设**: 直接内置 **Nature**, **Science**, **Cell** 三大期刊风格预设。一键应用符合各刊要求的字体（Helvetica/Roboto/Arial）、边框和样式。
- 📊 **深度统计分析**:
  - **显著性标注**: 自动执行统计检验，为柱状图/箱线图添加 p 值星号 (`*`, `**`, `***`)。
  - **趋势线**: 支持散点图/折线图线性回归拟合，并展示 $R^2$。
  - **误差棒**: 全球标准的 SD（标准差）/ SE（标准误）自动计算与绘制。
- 🎨 **权威配色系统**: 内置 Nature, Science, Lancet, NEJM, Cell 等主流学术期刊配色。
- 🛠️ **精细布局控制**:
  - **坐标轴间隔**: 自由控制轴刻度密度，解决标签重叠问题。
  - **防重叠标签**: 智能算法处理数值标签，确保文字不重合，保持在图形内部。
  - **学术宽高比**: 提供 1:1, 4:3, 3:2 (学术常用) 等固定比例。
- 🚀 **高性能数据处理**: 轻松处理数万行数据。支持 `.csv`, `.tsv`, `.txt` 上传或 Excel 直接粘贴。
- 📥 **导出与复现**: 支持 SVG 矢量图与高分辨率 PNG。导出文件纯净无水印，符合出版要求。

## 📖 用户手册

### 交互绘图流程
1. **上传数据**: 转到“数据”标签页，上传文件或粘贴文本。支持公式列设置。
2. **选择字段**: 在“参数”页选择 X 轴（维度）和 Y 轴（指标）。
3. **AI 辅助**: 尝试在顶部的快捷指令框输入 `Nature风格`。
4. **统计修饰**: 开启“显著性标注”或“趋势线”。
5. **精调导出**: 调整标题和轴标签，点击右侧的导出按钮下载。
