# SciPlot - 科研绘图工具

SciPlot 是一款专为科研人员打造的轻量级、响应式在线绘图工具。它支持多种科学图表类型，具备智能数据解析和高性能大数据处理能力。

🔗 **在线体验**: [https://sciplot.vercel.app/](https://sciplot.vercel.app/)

---

## ✨ 核心特性

- 🚀 **高性能数据处理**: 采用分页渲染技术，轻松处理数万行 CSV/TSV 数据而不卡顿。
- 🧠 **智能图表推荐**: 根据你选择的 X 轴（维度）和 Y 轴（指标）字段类型，动态推荐最合适的图表类型。
- 📊 **丰富图表类型**: 支持折线图、散点图、柱状图、面积图、饼图、雷达图、热力图、箱线图、小提琴图等。
- 🛠️ **自由字段映射**: 支持自由指定任意列作为 X 轴，多选或一键筛选（仅数值列）作为 Y 轴。
- 💾 **数据持久化**: 在切换图表类型时，自动保留用户上传或修改的数据，防止工作丢失。
- 🎨 **科研级配色**: 内置 Nature、Science、Lancet、NEJM 等权威期刊配色方案。
- 📥 **高质量导出**: 支持导出透明背景的 SVG 矢量图或高分辨率 PNG 图片。

## 📖 用户手册

### 第一步：准备数据
- 支持 `.csv`, `.tsv`, `.txt` 格式。
- 第一行为表头（列名），后续为数据。
- 也可以通过“数据”面板的“粘贴”功能直接从 Excel 复制数据。

### 第二步：上传与编辑
1. 点击左上角的 **“数据”** 标签。
2. 点击 **“上传文件”** 或在表格中手动编辑。
3. 如果数据较多，可以使用下方的分页器浏览。

### 第三步：配置绘图
1. 切换到 **“参数”** 标签。
2. **选择 X 轴**: 在“X 轴解析字段”下拉框中选择你的维度列（如：时间、样本名）。
3. **选择绘图字段**: 在“绘图数据字段”列表中勾选你想展示的指标。你可以点击“仅数值”快速筛选。
4. **参考推荐**: 查看面板顶部的“智能推荐”，点击推荐的按钮快速生成最佳预览。

### 第四步：美化与导出
1. 在下方配置标题、坐标轴名称、配色方案等。
2. 在右侧预览区确认效果。
3. 点击 **“SVG”** 或 **“PNG”** 按钮下载图片。

## 🛠️ 开发指南

### 运行环境
- Node.js 18+
- npm

### 启动开发服务器
```bash
# 安装依赖
npm install

# 启动
npm run dev
```

### 生产环境构建
```bash
npm run build
```
